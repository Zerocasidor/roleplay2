#frank `frank,naomi,dai`

-- ไดและนาโอมิเดินมาถึงตลาด

[ ท่านนาโอมิ ท่านจะไปซื้อร้านไหนหรอครับ ] ไดถามนาโอมิพร้อมเดินมองร้านรอบๆ --

["ฮื้มม.."]

<นาโอมิกวาดสายตามองร้านค้าต่างๆที่ตั้งอยู่เรียงรายข้างกันเป็นแถว เธอเดินเบียดเสียดกับผู้คนเป็นระยะๆ ก่อนที่จะมาหยุดยืนอยู่ตรงร้านตรงหน้า

["ที่นี่แหละค่ะท่านได"]

<เธอหันมายิ้มให้ ก่อนจะเดินเข้าร้านไปพร้อมกับสหายหนุ่มขององค์ราชา ทันใดนั้นเองที่กลิ่นหอมหวานของแป้ง นมและน้ำตาลเข้าปะทะจมูก

-- [ กลิ่นนี้ชวนให้น้ำลายไหลจริงๆเลยย ] ไดสูดดมกลิ่มอันหอมหวานที่ตลบอบอวลไปทั่วร้าน

[ นี่มันทาร์ตสตรอวเบอรี่นี้นา ] ไดตาลุกวาวเป็นประกายเมื่อได้เห็นขนมหวานที่ตนชอบมากๆ --

["ซื้อกลับไปมั้ยคะ"]

<นาโอมิเมื่อเห็นอีกคนทำหน้าตาลุกวาวนั้นก็อดหัวเราะออกมาไม่ได้ ก่อนจะเสนอให้อีกคนซื้อมันกลับไปกินที่วังกับทุกคน

["เอาหมดนี่เลยจ้ะคุณยาย"]

เธอกล่าวกับหญิงสาวเผ่าเอลฟ์วัยชรา เธอยิ้มเมื่อยายเจ้าของร้านยิ้มตายีมาให้ ก่อนที่เธอจะวางถุงผ้าพอดีมือ ข้างในมือเหรียญโกลด์ล้วนเต็มไปหมด จากนั้นก็ค่อยๆหยิบขนมหวานวางในตะกร้า

-- [ งั้นเอาอันนี้ อันนั้นด้วย แล้วก็อันนี้อีก ] ไดหยิบขนมหวานวางในตะกล้าเรื่อยๆจนพอใจจึงหยุดมือ

[ เอาไปเยอะเกินไปไหมนะ ] ไดกล่าวอย่างกังวลเล็กน้อย --

["ฮึฮึ... เอาไปตามใจอยากเลยค่ะ ไม่ต้องห่วง"]

<นาโอมิหัวเราะเอ็นดูในความตื่นเต้นของอีกคน ก่อนจะเอ่ยเพื่อให้อีกฝ่ายสบายใจลง

["ท่านไดหยิบขนมน่าทานหลายอย่างเลยนะคะเนี่ยย"]

-- [ แฮะๆ ผมคิดว่าท่านราชาน่าจะชอบนะครับ ] ไดกล่าวพร้อมยิ้มให้

[ ผมว่าแค่นี้น่าจะพอแล้ว ] --

----แฟรงค์เดินผ่านร้านขนมแห่งหนึ่ง แต่กลิ่นเลือดของเด็กน้อยที่บริสุทธ์และสารเคมีร้ายแรงที่เป็นพิษต่อธรรมชาติ กลับอบอวลเตะจมูกเขา----

["ชิ! แย่ละ คนทั่วไปอาจจะยังไม่รู้ตัว ฉันต้องรีบกลับไปที่แล็ปแล้ว"]

ทันทีที่แฟรงค์รู้ตัว เขามีท่าทีกระวนกระวายแรกน้อย ยังดีที่เขาใส่เสื้อคลุมเหมือนนักเวทย์ บางทีนี้อาจจะช่วยกลบเกลื้อนรอยเลือดบนตัวของเขาได้(?)

-- ในระหว่างที่นาโอมและไดกำลังพูดคุยกันอยู่ไดก็ได้กลิ่นที่แปลกออกไป

[ *นะ..นี่มันกลิ่นเลือดนิ* ] ไดได้มองไปยังต้นต่อของกลิ่นจึงพบชายปริศนาท่าทางน่าสงสัยที่คล้ายจะเป็นนักเวทย์หรืออะไรสักอย่าง

[  ท่านนาโอมิครับ ท่านก็รู้สึกตัวสินะครับ ] ไดสกิดเรียกนาโอมิในระกว่างที่นาโอมินั้นกำลังจ่ายตังอยู่ --

[".. ค่ะ ดิฉันก็ได้กลิ่น.. กลิ่นเลือดคาวเล็กน้อย..."]

<เธอตอบเสียงแผ่ว โสตประสาทของเธอที่รับรู้ได้ถึงกลิ่นที่โชยมาตามลมเบาๆชั่วครู่ แววตาของเธอจริงจังมากขึ้นก่อนจะหันไปมองต้นตอของกลิ่นเล็กน้อย

["อย่างแรกเรารีบออกไปจากร้านกันก่อนเถอะค่ะ"]

[“ให้ตายสิ กลิ่นเลือดเป็นทางขนาดนี้ ไม่มีทางเลือกแล้วสินะ”]

แฟรงค์รู้สึกได้ถึงสไลม์ที่น่าสนใจกับเอล์ฟที่จ้องมองอย่างน่าสงสัย จึงรีบวิ่งออกไปในทันที

-- [ *คิดจะหนีหรอ* ] ไดวิ่งตามชายที่น่าสงสัยไปโดยทิ้งนาโอมิไว้ที่ร้าน --

["!? รอเดี๋ยวค่ะท่านได!"]

<เธอรีบเตือนอีกฝ่ายที่กำลังพุ่งไปตามชายแปลกหน้า ทว่ากลับไม่ทันเสียแล้วเพราะสไลม์หนุ่มหายไปต่อหน้าต่อตาเธอ

["แย่จริง มีแต่ต้องวิ่งตามไปสินะ"]

<เธอรีบวิ่งออกจากร้านด้วยความเร็วสูง นาโอมิใช้ความคล่องแคล่วยืดหยุ่นของร่างกายเธอในการวิ่งฝ่าผู้คนที่อัดแน่นเบียดเสียดกัน กระทั่งดวงตาก็พบกับเด็กหนุ่มคุ้นตาเสียที พร้อมกับเอลฟ์ชราที่เรี่ยวแรงและการเคลื่อนไหวดูจะไม่ชราเท่าไหร่นักจนผิดสังเกต

-- [ เจ้าเป็นใครกัน ทำไมตัวเจ้าถึงมีกลิ่นเลือด ] ไดถามชายน่าสงสัยที่อยู่ต่อหน้าเขา --

[`“หืม… คงจะกลับไปที่แล็ปไม่ได้แล้วละ ”`]

แฟรงค์เมินคำถามของเด็กหนุ่มแปลกๆ และวิ่งลัดเลาะออกไปนอกเมืองหวังจะสกัดให้หลุด

["นั่น.. กำลังจะออกนอกเมืองสินะ"]

<นาโอมิเห็นทิศทางการเคลื่อนไหวของชายแปลกหน้าก็รู้ได้ เธอจึงใช้ร่างกายเบาหวิวปีนขึ้นไปบนหลังคาบ้านหลังหนึ่งก่อนจะตะโกนเบ่งออกมาไปทางประตูเมืองซึ่งมีทหารเผ่าเอลฟ์ประมาณสามสี่นายประจำการอยู่

["ทหารคะ!! จับตัวชายแก่คนนี้ที!!"]

<สิ้นสุรเสียงหวานแหบ เหล่าทหารที่ประจำการอยู่ก็ต่างมองไปทางที่นิ้วของนาโอมิชี้ ก่อนจะหยิบอาวุธเตรียมตั้งท่ารับทันที

แฟรงค์วิ่งจนมาถึงประตูเมืองแต่กลับสลัดคนที่ไล่ตามเข้ามาไม่ได้

[“เฮ้ย! หยุดเดี๋ยวนี้นะ!”]

เสียงยามเฝ้าประตูที่ใส่ชุดเกราะหนักจนไม่เห็นหน้า ยกหอกเหล็กขึ้นมากันทางเอาไว้ แต่แฟรงค์กลับสไลด์ลอดใต้ออกไปได้อย่างหวุดหวิด

[“โว้ว เกือบไปแล้ว”]

["แฮ่ก...! จับเขาได้มั้ยคะ"]

<องค์ราชินีเอลฟ์วิ่งมาหยุดอยู่หน้าป้อมปราการประตูเมือง เธอหันมองซ้ายขวาอย่างถี่ถ้วน ทว่าก็ไม่พบชายแปลกหน้าผู้นั้นเลยแม้แต่ปลายเส้นผม เธอหันไปมองไดอย่างมั่นใจ ก่อนที่เธอจะวิ่งออกไปทันทีพร้อมสไลม์หนุ่ม

["เดี๋ยวขอรับท่านราชินี!!"]

เสียงแหบห้าวจากเหล่าทหารที่ยืนประจำการไล่หลังนาโอมิมา แต่เธอก็ไม่สนใจพร้อมเสกหอกน้ำแข็งยาวในมือขึ้นมาถือไว้

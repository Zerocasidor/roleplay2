'["ข้างนอกดูวุ่นวายจังเลย..."]
< หญิงสาวเอ่ยเสียงแผ่ว ดวงตาสีชมพูสว่างเลื่อนมองออกไปยังนอกหน้าต่าง นาโอมิซึ่งได้ถูกเลื่อนขั้นจากคู่หมั้นขององค์ราชาแห่งราชอาณาจักรแห่งเผ่าเอลฟ์ กลายมาเป็นราชินีผู้ปกครองบ้านเมืองร่วมกับพระราชา มือข้างหนึ่งที่ยังคงงดงามเสมอแม้จะถูกใช้งานมาแล้วนับครั้งไม่ถ้วนสัมผัสเข้าที่บานหน้าต่างใสสะอาดที่เริ่มมีหิมะปกคลุม ความเย็นที่เกาะกินปลายนิ้วมือของเธอทำให้เธอต้องชักมือกลับ ในระหว่างทางเดินของห้องโถงยาวสีขาวสะอาดดั่งหิมะบริสุทธิ์ที่ไร้ผู้คนกลับมีเสียงๆหนึ่งค่อยๆดังขึ้นมา

["แอ๊! อุแว๊!"]

< เป็นเจ้าสิ่งมีชีวิตก้อนน้อยๆ ถูกพันด้วยผ้าไหมพรมที่ถูกถักทอมาอย่างประณีตและละเอียดจากราชินีผู้งดงาม ที่กำลังร้องไห้ออกมาด้วยน้ำเสียงอ่อน ราวกับสัมผัสได้ว่าอุณหภูมิของผู้เป็นมารดาเริ่มลดลง

["โอ๋ๆ ไม่ร้องนะจ๊ะเจ้าหญิงของแม่... หิวนมเหรอจ๊ะ..."]

< เธอสะดุ้ง รีบกอดทารกน้อยในอ้อมแขนเล็กของตนเองช้าๆแล้วกล่อมให้ลูกน้อยอารมณ์ดีขึ้น ก่อนที่ขาเรียวสวยจะค่อยๆเดินไปตามพรมแดงจนไปถึงประตูทางเข้าสีน้ำตาลห้องๆหนึ่งที่มีขนาดใหญ่โต ประดับประดาด้วยกรอบทองและฝังอัญมณีสีเขียวอันเป็นสีสัญลักษณ์ของอาณาจักรของเผ่าพันธุ์ที่ใช้ชีวิตกับป่าไม้และธรรมชาติอันเขียวชอุ่ม

แต่บัดนั้น ในอนาคตอาจจะเหลือเพียงแค่เศษขี้เถ้าของใบไม้ที่ถูกไฟแผดเผาจนเหลือเพียงแค่ตอตะโก

["ขออนุญาตค่ะ..."]

< นาโอมิกำมือเล็กซีดของเธอแล้วออกแรงเคาะประตูให้เกิดเสียงดัง ก่อนจะเปล่งวาจาออกไปด้วยน้ำเสียงอ่อนหวานราวกับเสียงดีดพิณ

< มือเรียวค่อยๆผลักประตูหนักเข้าไปภายในตัวห้องประชุมขนาดใหญ่ ภายในตกแต่งด้วยของหรูหรา มีทั้งแก้ว เครื่องเคลือบ และอัญมณีต่างๆประดับอยู่ แต่นั่นก็ไม่ใช่สิ่งที่เธอต้องการจะสนใจ

หญิงสาวแปลกใจมากขึ้นเมื่อเจอแขกผู้มาเยือนอีกท่านหนึ่ง นาโอมิได้ยินมาจากผู้เป็นสามีของตนเองว่าจะเรียกคนรู้จักมายังปราสาทสวยแห่งนี้ ดวงตาสีชมพูสวยสว่างสบตากับดวงตาสีสวยของชายหนุ่มร่างสูง เมื่อนั้นเธอก็เผยรอยยิ้มหวานออกมาบนใบหน้ากลมเล็ก

"[ท่านเซน! ดีใจจังเลยค่ะที่ได้พบท่านอีก!"]

< เธอเผลอเอ่ยออกไปโดยไม่ได้มองถึงภาพรวมของสถานการณ์ในตอนนี้ ดังนั้นเมื่อเธอก้าวผ่านธรณีประตูมาสองก้าวก็สัมผัสได้ถึงแรงกดดันบางอย่าง องค์ราชาอย่างฟีลและเซนในตอนนี้ดูเคร่งเครียดอย่างมาก เธอจึงจับชายกระโปรงสีชมพูแต่งแต้มด้วยลูกไม้สีขาวสลับดำขึ้นแล้วย่อตัวลงเป็นเชิงเคารพแกมขอโทษ

["ขอประทานอภัยด้วยเพคะท่านราชา ดิฉันไม่ได้ทันสังเกตเสียนี่" ]

『  [" อะ  สายันสวัสดิ์ ครับนาโอ "]
คนงามรีบเปลี่ยนสีหน้าก่อนวาดยิ้มบางบนริมฝีปากทักทาย หญิงสาวที่มีศักด์เป็นราชินีตรงหน้า

ดวงแก้วสีอำพันพลันเป็นประกายเมื่อเหลือบไปเห็นเจ้าก้อนสิ่งมีชีวิตในอ้อมแขนของนาโอมิ ทารกเพศหญิงที่มีน่าตาน่ารักจิ้มลิ้มดูน่าทนุถนอมกำลังนอนหลับไม่รู้เรื่องอยู่ในอ้อมแขนเล็กของผู้เป็นมารดา 

คนงามก้มลงเล็กน้อยพร้อมส่งยิ้มนวลให้กับทารกตัวจิ๋ว 

[" สวัสดีครับองค์หญิง วันนี้หนาวรึปล่าว ? "]

มือเรียวยกขึ้นสัมผัสพวงแก้มเล็กอย่างแผ่วเบาครั้นกลัวเจ้าตัวเล็กจะเจ็บจากการกระทำของตน  』 


["ฮึฮึ อุ้มได้นะคะคุณเซน หนูโนเอลคิดถึงคุณเซนมากๆเลย"]

< เธอว่าพลางค่อยๆประคองทารกน้อยในอ้อมกอดตนแล้วยื่นให้อีกฝ่ายช้าๆอย่างนุ่มนวล พลางมองเด็กน้อยดวงตาสีชมพูเดียวกับตนกลมโตที่กำลังยิ้มร่าเริงเมื่อได้เจอกับชายหนุ่มที่หนูน้อยคุ้นตา

["แอ๊ แอ๊!"]

< แขนป้อมๆเล็กยื่นออกมาแล้วใช้ฝ่ามือเล็กนิ่มแปะบนแก้มของหนุ่มร่างสูงผมยาวสีดำขลับ พลางยิ้มหัวเราะชอบใจที่ได้จับแก้มของคนตรงหน้า เมื่อนาโอมิเห็นดังนั้นก็ยิ่งยิ้มเอ็นดูลูกสาวตัวน้อยของตนที่เพิ่งเกิดได้ไม่กี่ปี


『  เมื่อสัมผัสนุ่มนิ่มของทารกแตะลงบนแก้มขาวของคนงาม เจ้าของแก้มขาวที่ถูกสัมผัสพลันสะดุ้งเล็กน้อย แล้วจึงคลี่ยิ้มให้เจ้าหญิงน้อยที่อยู่ในอ้อมแขนของตน

[" นี่โนเอลกำลังเอาคืนผมอยู่หรอครับ ฮึ ? "]

คนงามอดคิดไม่ได้ว่าการที่ตนถือวิสาสะไปหยุมแก้มองค์หญิงน้อย จะโดนทารกตัวจิ๋วหยุมแก้มเอาคืนเช่นนี้   』 


< นาโอมิที่ได้ยินคำพูดของชายหนุ่มรูปงามร่างสูงสมส่วนกล่าวก็อดหัวเราะออกมาเบาๆไม่ได้ การกระทำที่เซนและโนเอลได้ทำต่อกันมันช่างน่ารักในสายตาของหญิงสาว หัวใจที่เย็นเฉียบด้วยความโศกเศร้าเล็กน้อยก็กลับมาสดใสดังเดิม เธอรู้สึกอบอุ่นมากๆ จึงเอ่ยกับอีกฝ่ายเล็กน้อย

["โนเอลติดคุณเซนขนาดนี้ ฉันคงต้องพาไปเที่ยวที่ร้านหนังสือบ่อยๆแล้วล่ะค่ะ ฮึฮึ"]

< นาโอมิเอ่ย ก่อนมือข้างหนึ่งจะล้วงหาถุงบางอย่างในกระเป๋าจิ๋วที่พกมาด้วย นิ้วเรียวสวยหยิบสิ่งของบางอย่างออกมาจากช่องกระเป๋าของเธอ

มันคือผ้าเช็ดหน้าสีขาวอ่อนลายลูกไม้สีชมพูสวย มีกลิ่นหอมของดอกไม้อ่อนๆโชยออกมาจากผ้าผืนเล็ก อีกผืนเป็นสีม่วงอ่อนๆ มีกลิ่นหอมอ่อนของโคโลญจน์เช่นกัน เธอยื่นให้กับชายหนุ่มร่างสูงโปร่งอย่างสุภาพก่อนจะส่งยิ้มให้อีกฝ่าย

["ฉันเย็บผ้าเช็ดหน้าผืนนี้ให้กับคุณเซนและคุณเจดโดยเฉพาะเลยค่ะ อาจจะไม่มากมายนักเท่าไหร่ พวกคุณเอาไปใช้ได้ตามสบายเลยนะคะ"]

< หญิงสาวเผ่าเอลฟ์เอียงคอเล็กน้อยแล้วเอ่ยอย่างร่าเริง ก่อนจะจับกระโปรงสีชมพูยาวของตนเองขึ้นแล้วย่อตัวลงโค้งเป็นเชิงขอบคุณอีกคน

『     [" หุหุ ร้านของผมยินดีต้อนรับเสมอครับ "]

คนงามกล่าวขึ้นด้วยความยินดี   การที่ร้านหนังสือที่เงียบเหงาของเขาจะมีสีสันขึ้นมาบ้าง เพราะจะมีแขกผู้น่ารักมาเยี่ยม ถือว่าเป็นข่าวดีสำหรับเขายิ่งนัก


เมื่อฝ่ามือเนียนสวยของหญิงสาวตรงหน้ายื่นผ้าเช็ดหน้าสีขาวอ่อนลายลูกไม้ ที่มีกลิ่นหอมนวลเฉพาะตัวมาให้นั้น  ใบหน้างามของชายหนุ่มพลันฉายแววประหลาดใจ ไม่มีคิดว่าตนจะได้ของขวัญที่ใส่ใจเช่นนี้

[" ขอบคุนมากเลยครับ ผมจะเก็บอย่างดีไม่ใช้เลย ฮะฮะ "    』]



『  เหมือนเพิ่งนึกอะไรได้ คนงามรีบคว้าควานหานาฬิกาพกที่อยู่ในกระเป๋าเสื้อโค้ทสีดำที่ดูหรูหราของตน ก่อนที่เหลือบมองเวลาในนาฬิกาพกแล้วจึงเงยหน้าขึ้นมา


[" กระหม่อมคงต้องขอตัวลาเสียแล้ว พอดีมีธุระที่ต้องสะสางน่ะครับ "]

เซ็นกล่าวอย่างร้อนลนก่อนที่จะขอตัวกลับก่อน  』 

["โอ้ ได้สิคะ ถ้างั้นกลับดีๆนะคะคุณเซน รักษาสุขภาพด้วย"]

< หญิงสาวพยักหน้าตอบรับอีกฝ่ายเบาๆแล้วเอ่ยไล่ตามหลังอีกคนที่เดินออกไปไกลมากขึ้น พลางกระชับอุ้มลูกสาวของเธอไว้ซึ่งในตอนนี้เด็กน้อยเริ่มจะเข้าสู่นิทราแล้ว

["งั้นพาเจ้าหญิงน้อยไปนอนก่อน แล้วไปหาฟีลแล้วกันนะ"]

< เธอเอ่ยเบาๆกับตนเอง แล้วจุมพิตเข้าที่หน้าผากมนๆของทารกน้อยก่อนจะเดินออกจากห้องประชุมใหญ่โอ่อ่านี้ไป

["โอ๋เอ๋.. อะ ไลอาคะ.. ฝากเด็กคนนี้ทีได้มั้ยคะ พาเขาเข้านอนแทนข้าที... ข้ามีธุระต่อน่ะค่ะ"]

["ได้เพคะฝ่าบาท... ดิฉันขออนุญาตอุ้มองค์หญิงไปนอนพักผ่อนบนเปลนะเพคะ"]

< เมื่อราชินีแห่งเผ่าเอลฟ์เปิดประตูมาถึงห้องบรรทมสีชมพูอ่อนๆดูสบายตา ดูก็สามารถมองออกได้ว่าเป็นห้องของเด็กผู้หญิง หรือก็คือเป็นห้องทารกน้อยในอ้อมแขนเธอซึ่งหลับปุ๋ยอยู่ในตอนนี้ หญิงสาวกวาดสายตามองหาบุคคลที่ต้องการด้วยความเฉียบคม ทำให้เจอกับสาวใช้คนสนิทที่รับหน้าที่ดูแลองค์หญิงน้อยของอาณาจักรแห่งนี้อยู่ 'ไลอา แรนเชส'

["ฝ่าบาทเพคะ ดิฉันไม่ทราบว่าฝ่าบาทต้องการจะไปที่ใด แต่ข้างนอกเริ่มมีหิมะตกหนัก ได้โปรดสวมเสื้อให้อุ่นร่างกายไว้และพยายามอย่าฝืนร่างกายของฝ่าบาทนะเพคะ"]

< หญิงสาววัยกลางคนผู้เป็นสาวใช้ประจำกายของท่านราชินีเลยก็ว่าได้ ได้กล่าวแก่ผู้มียศใหญ่กว่าตนด้วยความเป็นห่วงที่ผ่านออกมาทั้งน้ำเสียง คำพูดและสีหน้าของเธอ ก่อนที่จะอุ้มรับเจ้าหญิงเอลฟ์ตัวน้อยจากนาโอมิด้วยความระมัดระวังและทนุถนอมนุ่มนวลมากที่สุด ก่อนจะค่อยๆหันหลังเดินไปวางทารกหญิงแก้มจิ้มลิ้มไว้ในเปลสีขาวสะอาด ประดับด้วยระบายสีชมพูที่ตัดเข้ากันอย่างพอดิบพอดี หลังจากนั้นจึงหยิบตุ๊กตาหมีสีขาวฝีมือมารดาของเด็กน้อยมาไว้ข้างๆเผื่อองค์หญิงจะมีฝันที่ดี

["เฮ้อ หายไปไหนของเขานะ..."]

< ขาเรียวยาวสวยบูทยาวสีดำส้นสูงกระทบกับพื้นหินอ่อนเงาวับที่ถูกขัดมาอย่างสะอาดเกิดเสียงดังเบาๆ มือขาวนวลเรียวสวยทั้งสองยกกระโปรงของตนขึ้นเพื่อให้เคลื่อนไหวได้สะดวกมากขึ้น นาโอมิหันซ้ายขวาช้าๆ ดวงตาสีชมพูสว่างกำลังเลื่อนหาบุคคลเรือนผมสีขาวสวยราวก้อนเมฆด้วยสายตาอันเฉียบคมของเธอ แต่แม้จะเดินตั้งแต่หน้าทางเข้าของพระราชวังจนสุดถึงสวนดอกไม้หลังพระราชวังก็ไม่พบบุคคลที่เธอต้องการพบเลยแม้แต่ปลายเล็บ

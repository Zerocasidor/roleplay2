| [" ... "] ฟีล ได้เดินเข้ามาในป่าลึกก่อนจะปล่อยรังสีสังหารอย่างรุนแรงออกมาด้วยความโกรธก่อนจะเริ่มฝึกสังเคราะห์อาวุธของตนที่ไม่ได้นำมาใช้เป็นหลายทศวรรษ

[" คุไนย....อา....ช่างน่าคิดถึง ]" ราชาผู้หนึ่งที่เริ่มสังเคราะห์อาวุธตนมาได้อย่างสมบูรณ์แบบ

หลังจากนั้นในระยะเวลาไม่ถึงชั่วโมง ฉายา เงาสีเลือด ก็ได้กลับหวนคืนสู่ชายผู้นี้อีกครั้ง |

| [" ชิ... "] ฟีล ได้เดินเข้าไปเรื่อยๆในป่า เสียงเดินของฟีลที่เหยียย่ำบนผืนหญ้าในป่าอันสวยงาน เป็นเสียงที่ทำให้รู้สึกคนละขั้วกับป่าสีสันสดใส ตัวตนของเขานั้นมืดหม่น รังสีของเขาเหมือนไม่ใช่คนดี

ไม่ใช่ทั้งราชา ไม่ใช่ทั้งนักฆ่า เรียกว่าอย่างไรดี

....ยมทูตหรอ....ไม่เลวเลยนะ เปรียบเป็นยมทูตที่จะส่งมอบประสงค์ของผู้ที่อยากมีชีวิตรอด....ให้ตายอย่างช้าๆและทรมาณ

ชายผู้นี้ อยู่ในป่าเพียงลำพัง ป่าลึกนั้นขึ้นชื่อว่าเป็นป่าที่มีมอนส์เตอร์เยอะ แต่หาเขาเกรงกลัวไม่

มีสัตว์หลายชนิดเดินผ่านชายผู้นี้แต่ก็วิ่งหนีไปให้ไกลแทบทั้งสิ้น

เขา ไม่สิ....ชายผู้นี้เป็นสิ่งมีชีวิตที่เหนือสิ่งมีชีวิต มีศัตรูมากมายเดินจะมาโจมตีเขา แต่ก็ถูกดูดเข้าไปในก้อนสีดำบางอย่างที่ลอยอยู่รอบๆตัวเขาในทันที

อะไรหน่ะ.....หลุมดำหรอ...?

อ่อใช่....สิ่งมีชีวิตตนนี้เป็นผู้ใช้พลังหลุมดำ และสัตว์อัญเชิญ

ข้าไม่เคยได้ยินมาก่อนเลยว่าบนโลกนี้จะมีพลังหลุมดำอยู่ ถ้าจะคล้ายๆก็คงเป็นความมืดแปรผันสินะ?

คงจะทำนองนั้น

| [" เกะกะ.. "] ยมทูตสีดำทมิฬในป่าได้ยิงก้อนหลุมดำเล็กๆผ่านนิ้วชี้ที่ชี้ไปด้านข้างเพื่อยิงสัตว์ประหลาดทางด้านซ้ายมือ

ดูจะเป็นก้อนสีดำเล็กๆ แต่จู่ๆ ก้อนสีดำเล็กๆนั้นทำให้สัตว์ประหลาดตัวเป็นรูขนาดใหญ่เหมือนโดนยิงพลังสักอย่างที่มีความรุนแรงสูง

ห....โหดร้ายเกินไปแล้ว เจ้านั่นแค่มองมาเองนะ!! นี่ เจ้าอย่าพังป่าสิ๊!

ห้ามไม่ได้หรอกนะ.. |

| ว่าแต่ว่าเขาจะลองอะไรในป่าหรอ?

ยมทูตตนนั้นได้นำคุไนยออกมาแล้ววิ่งไปด้วยความเร็วสูงกก่อนจะวิ่งไปกระโดดเตะท้องของออคสีเขียวตัวใหญ่ที่ดูหน้าสะพรึง

หลังจากที่กระโดดเตะท้องของมันเสร็จแล้วก็โยนคุไนยเชื่อมโซ่เล่นหนึ่งไปปักกับท้องของมันอีกครั้งจนทะลุก่อนจะเตะตัดขาของออคตัวใหญ่จนล้มลง

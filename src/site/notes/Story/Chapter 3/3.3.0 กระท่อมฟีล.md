ปรากฎเห็นเป็นชายหนุ่มรูปร่างงดงาม มีรอยแผลบางส่วนบนใบหน้าดูเรียบง่ายและเป็นแผลที่ดูจะใหญ่และสาหัส

ราชาแห่งราชอาณาจักรเอลฟ์ได้ค่อยๆมีรอยปานเลอะเหมือนหมึกบ่งบอกว่าท่านราชาระแวงอยู่พอสมควร |

| ["????"] ชายหนุ่มคนนั้นได้งงงวยแล้วพูดถามขึ้นมาว่า ที่นี่ที่ไหนแล้วเจ้าเป็นใครกัน |

|[" ใครกัน "] ฟีลได้ปล่อยตะเกียงไปกดตัวอีกคนไว้ด้วยแรงโน้มถ่วงมหาศาลด้วยใบหน้าที่อารมณ์โกรธที่มาพังบ้านอันแสนรักของตน |

// [" เจ้านั่นแหละเป็นใครแล้วนี่ที่ไหน "] ดันได้ดันตัวเองขึ้นแล้วหลุดออกมาจากพลังตะเกียงของราชาเอลฟ์ แล้วได้ใช้การตวัดดาบทำให้ราชาเอลฟ์ล้มลง"

| ฟีล ได้ใช้มือค้ำตตัวตนไว้แล้วเอาตะเกียงมาบังไว้แล้วเปลี่ยนเป็นดาบขนาดยาวมากันคลื่นดาบอีกฝ่ายไว้ ก่อนจะตีลังกาวนไปด้านหลังกลับมาแล้วเดินไปดึงดาบที่ปักพื้นอยู่

ยมทูตสีดำในคราบราชาเอลฟ์ครึ่งปีศาจได้ยกดาบขึ้นชี้ใส่ชายตรงหน้าก่อนจะพุ่งเข้าไปด้วยความเร็วสูงแล้วใช้ดาบตวัดไปด้านหน้าด้วยวงระยะดาบทำให้ฟาดโดนอีกฝ่ายเต็มแรง |

| [" อึก "] ฟีลได้สัมผัสถึงคลื่นดาบนั้นที่ผ่านตัวไป แต่พอรู้ตัวอีกทีก็เห็นได้ถึงร่างของฟีลได้กลายเป็นเงา ปรากฎอีกทีเป็นฟีลที่ยืนอยู่อีกด้านหนึ่งข้างนอกป่าก่อนจะกล่าวขึ้น

[" อสรพิษ....กลืน...กิน.."] ฟีลย่อขาทั้งสองข้างลงมาเล็กน้อย ยื่นมือไปด้านหน้าก่อนจะกำใส่ชายด้านหน้าตนก่อนจะปรากฎบางสิ่งขนาดใหญ่ออกมา

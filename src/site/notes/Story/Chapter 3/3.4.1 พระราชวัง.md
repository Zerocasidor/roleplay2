#feel `feel,dai`

-- ก๊อกๆ ก๊อกๆ เสียงเคาะประตูดังขึ้นที่หน้าห้องของราชาเอลฟ์

[ ท่านราชา ท่านราชา ] เด็กหนุ่มส่งเสียงเรียกราชาเอลฟ์ที่หน้าประตู

[ ไปสำรวจในป่ากันเถอะ ท่าราชา! ] ไดกล่าวด้วยท่าทางลุกลี้ลุกลน --

| [" หือ? อื้อ เข้ามาสิ "] พูดด้วยน้ำเสียงอู้อี้ขณะกำลังรวบผมยาวๆของตนอยู่

ดูเผินๆก็เหมือนผู้หญิงเหมือนกัน

ที่ปากของพระราชามียางมัดผมที่ราชาคาบอยู่เตรียมจะรวบ |

-- [ ขออนุญาตนะค้าบบ ] เด็กหนุ่มเปิดประตูเข้าไปเผยให้เห็นร่างของท่านราชาที่สง่างามราวกับหญิงสาว

เด็กหนุ่มยืนตลึงไปสักพักนึง แล้วจะกล่าวถามท่านราชา [ ท่านราชา พวกเราไปสำรวจป่ากันดีไหมครับ ] --

| [" ได้นะ "] พยักหน้าก่อนจะค่อยๆทำทุกอย่างให้เสร็จก่อนจะเดินไปหาอีกฝ่ายในท่าเตรียมพร้อม |

-- [ งั้นก็ออกเดินทางได้! ] เด็กหนุ่มเดินนำท่านราชาไป ทว่า

[ เอ่อออ....มันไปทางไหนอะครับ แฮะๆ ] เด็กหนุ่มผู้ไม่รู้ทางจึงหันกลับไปถามท่านราชาด้วยน้ำเสียงขี้เล่น --

| [" มานี่มามะ ]" ฟีลกล่าวก่อนจะเดินไปหาอีกฝ่ายแล้วเปิดประตูด้านหลังอีกคนก่อนเดินออกไป

[" นาโอมื! ข้าไปป่าก่อนนะ! "] ฟีลตะโกนเรียกลั่นปราสาทก่อนจะเดินออกไปกับได |

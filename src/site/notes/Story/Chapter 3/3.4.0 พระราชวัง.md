--------หลังจากที่เจ้าสไลม์สลบไป ฟีลได้เอากลับที่ปราสาทไปให้หลายๆคนดู--------
| [" เอาไงกับไอก้อนกลมๆนี่ดี "] ฟีลกล่าวกับดันที่ยืนกอดอกอยู่ข้างๆดันที่กอดอกสงสัยว่าต้องทำไงต่อดีเหมือนกัน

[" อ้ะ เอามันไปย่างไฟแล้วจิ้มเกลือกินกันมั้ย "] ฟีลหันมาถามดันเหมือนจะคิดอะไรออก |

// ดันได้ตอบราชาเอลฟ์กลับไปว่า 
["ข้าว่าเจ้าก้อนกลมๆนี่มันกินไม่ได้หรอกนะเจ้าหูยาว... มันแปลงร่างเป็นคนได้ด้วยข้าว่าเจ้าไม่ควรกินมัน หรอกนะ "]  
["ข้าว่าเจ้าควรจะคุยกับมันมากมากกว่านะว่าทำไมมันถึงมาอยู่ที่นี่ "]

| [" ก็ถูก "] ฟีลพูดก่อนจะนั่งลงเชิงเซ็งๆก่อนจะเอาเท้าสะกิดสไลม์ตรงหน้าเบาๆเรื่อยๆเชิงปลุก |

| [" กลัวอะไรขนาดน้านไอหนุ่ม "] เอาเท้าลงจากโต๊ะก่อนจะกอดอกจ้องไปที่อีกฝ่ายแล้วตบเบาะให้มานั่งใกล้ๆ |

-- ไดเดินไปด้วยความกลัว แล้วนั่งข้างๆฟีล --

| [" กินไรป่าว "] ฟีลถามขึ้นอย่างเป็นกันเองก่อนจะใช้โทรศัพท์บ้านโทรหาเซ็น 

[" โหลๆ เทสๆ เซ็น เรียกนาโอมิให้หน่อยสิ "] สิ้นสุดคำพูดที่อีกฝ่ายรับสาย ท่านราชาก็ตัดสายทิ้งทันทีก่อนจะกลับไปนั่งข้างๆทั้งสองเช่นเคย |

<เสียงเดินกระทบกับหินอ่อนช้าๆ มือเรียวผลักประตูบานใหญ่ช้าๆขณะเดินเท้าเอวมายืนตรงหน้าห้อง

["ฝ่าบาทคะ"]

["กรุณาเวลาเอาใครเข้าวังมาช่วยบอกดิฉันสักครั้งได้หรือไม่คะ"]

<เธอขมวดคิ้วนิ่วหน้าขณะเอ่ยด้วยน้ำเสียงเน้นกัดฟันพูด

-- ไดมองหน้าหญิงสาวด้วยความกลัว แล้วจึงหลบหน้าหนี

[ *คนบ้านนี้น่ากลัวชะมัด* ] --

| [" โอ้ะ...โทดทีครับ "] พนมมือข้างหนึ่งเชิงขอโทษพรางด้วยอีกมือหนึ่งกำลังจิกหัวสไลม์มาดึงหน้าใส่อีกฝ่ายให้พนมมือสวยๆสวัสดีภรรยาของตน |

-- [ สะ...สวัสดีคะ...ครับ ] พูดด้วยน้ำเสียงสั่นกลัว --

["เอ่อ..."]

["สวัสดีค่ะคุณ.."]

<หญิงสาวเลิ่กลั่กเล็กน้อยพลางพนมมือสวัสดีตอบอย่างงุนงง แล้วเอียงคอเล็กน้อยก่อนจะยิ้มหวานให้คนข้างๆอีกฝ่าย

["คุณดันคะทานอะไรสักหน่อยมั้ยคะ ดิฉันจะไปรินชาให้"]

<เธอโค้งคำนับให้กับชายผมยาวประบ่าช้าๆก่อนจะเดินไปรินชาที่เคาท์เตอร์ข้างๆโต๊ะที่ทั้งสามนั่งก่อนจะเดินมาพร้อมกับแก้วชาสละสลวยสีขาวนวลขอบทองสามแก้วก่อนจะวางลงช้าๆ

ดันได้กล่าวขึ้น ["มีเหล้าให้ข้าหรือไม่ขอรับ.... พอดีข้าไม่ค่อยถูกกับไอ่สิ่งที่เรียกว่าชา ซัก เท่าไหร่..."]

| [" อร่อยออก "] ฟีลกล่าว

["อะเหล้าเหรอคะ ดิฉันมีนะคะเดี๋ยวจะรินมาให้"]

<เธอเอ่ยดังนั้นก็รีบกุลีกุจอเดินออกไปหน้าประตู แต่ก่อนนั้นเธอหันมาถามผู้เป็นสามีของตน

["อยากกินอะไรคะฟีล... ฉันจะได้ไปทำเผื่อมาให้"]

["เป็นไปได้ก็ขอขนมครับ ]" ฟีลบอกกับนาโอมิที่น่าตาสดใสเบาๆอยู่ |

-- [ เอ่อ....ผมขอโอวันตินกับขนมแทนได้ไหมครับ ] ไดกล่าวต่อ --

["โอวันติน... นมช็อกโกแลตสินะคะ"]

<เธอหัวเราะเบาๆให้ก้อนสไลม์ข้างๆฟีลแล้วยิ้มให้ก่อนจะพยักหน้าให้ทั้งสาม

["งั้นวันนี้ฉันจะทำพายแอปเปิ้ล คุกกี้กับสลัดผลไม้เชื่อมให้แล้วกันนะคะ อยากทานเค้กส้มด้วยมั้ยคะ"]

<เธอถามทั้งสามคนอย่างสงสัย ใบหน้าหวานสวยยังคงรอยยิ้มกว้างบนใบหน้าไม่จางหาย

-- ไดยิ้มและพยักหน้าอย่างดีใจ --

<สักพักหญิงสาวก็เดินกลับมาพร้อมถาดที่มีขนมเรียงรายวางบนโต๊ะเต็ม พลางวางแก้วนมช็อกโกแลต เหล้าหนึ่งแก้วและน้ำแอปเปิ้ลเสริมต่อ

["เสร็จแล้วค่ะทุกคน ทานให้อร่อยนะคะ"]

<พูดจบเธอก็ส่งยิ้มบางให้ทั้งสาม เสร็จจากนั้นจึงนั่งลงบนเก้าอี้รับรองที่อยู่ห่างจากทั้งสามเล็กน้อยเพื่อรอคำสั่งจากผู้เป็นสามีของตน

-- ไดหยิบขนมขึ้นมา และทานอย่างเอร็ดอร่อย 

[ ขนมอร่อยมากเลยครับ ] ไดมีความสุขทุกครั้งที่ได้กินขนม

ไดหันไปหาภรรยาของฟีลแล้วยิ้มให้เล็กน้อยเพื่อแสดงความขอบคุณ --

["ดีใจจังเลยค่ะ ทานเยอะๆเลยนะคะ"]

<ได้ยินเพียงแค่นั้นเธอก็ยกมือทาบตรงหน้าอกแล้วโค้งตอบเบาๆด้วยความขอบคุณ

["มีอะไรต้องการอีกก็บอกดิฉันได้นะคะ"]

-- [ ที่นี้คือที่ไหนหรอครับ ทำไมมันใหญ่จัง ] ไดถามด้วยความสงสัย พลางยกนมช็อกโกแลตขึ้นมากินไปด้วย --

| [" ปราสาทคนแถวนี้แหละ "] ฟีลเอ่ยขึ้นกลางวงสนทนาขณะกินพายหยุบหยับในปาก |

["เสียมารยาทนะคะฝ่าบาท"]

<นาโอมิบุ้ยปากดุเล็กน้อยก่อนจะยิ้มให้แล้วหันไปหาแขกผู้มาเยือนยังปราสาทสวยของตน

["ที่นี่คือพระราชวังหลวงของอาณาจักรเอลฟ์ค่ะ พวกเราเป็นผู้นำของที่นี่ เหล่าคนรับใช้จึงจัดเตรียมสถานที่แสนวิเศษนี้ให้พวกเรา แต่พักนี้ท่านราชาของอาณาจักรเองก็แทบไม่ได้อยู่กับที่ห้องทำงานตัวเองเลยค่ะ"]

<เธอเอ่ยแนะนำสถานที่แห่งนี้ให้แขกผู้เป็นสไลม์ฟัง ก่อนเธอจะแตะแก้มตนแล้วหันมองสามีของตนที่กำลังกินพายอย่างเอร็ดอร่อยตรงหน้าด้วยแววตาที่ฉายแววเป็นห่วง

| [" งืออ ขอโทษ.. อะแฮ่ม..ข้าก็คือราชาที่นี่นั่นแหละนะ... "] ฟีลกล่าวขึ้นหลังกระแอมเล็กน้อย

ท่านราชาได้เอนตัวไปด้านหลังแบบท่าทงที่สบายขึ้น

[" นี่คืออาณาจักรเอลฟ์ เป็นอาณาจักรที่เผ่าพันธ์อื่นๆอยู่รวมกันอย่างสันติสุข...ค่อนข้างให้เลย เมืองนี้ตั้งอยู่ใจกลางป่า "] ท่านราชากล่าวละเอียดเล็กน้อยขณะมือก็กำลังวาดเป็นรูปอาณาจักรเล็กน้อยเป็นวงกลม |

-- [ *ก็สมกับเป็นราชาละนะ ก็เก่งขนาดนั้นเลยนิ* ] 

ไดลุกขึ้นและเดินไปหาภรรยาของฟีล [ นี่ๆ ข้าอยากเดินดูรอบๆน่ะ จะได้รึป่าว ] 

ไดถามด้วยท่าทางตื่นเต้นสุดๆ --

| [" เห็นอย่างงี้แม่นางของข้าก็เก่งมากๆเลยนะ "] ยิ้มเบาๆก่อนจะมองไปที่ภรรยาแสนสวยของตน |

["ชมไปก็ไม่ได้อะไรหรอกนะคะฝ่าบาท"]

<เธอลุกขึ้นก่อนจะดีดหน้าผากอีกคนเบาๆเล็กน้อยเป็นการหยอกเย้า

["ไหนๆก็พาแขกท่านนี้มาเที่ยวดูวังของเราสิคะ ดิฉันจะเก็บข้าวของตรงนี้เองฝ่าบาทพาท่านผู้นี้ไปเถอะค่ะ"]

<นาโอมิยิ้มแย้มให้ผู้เป็นสามีของตนก่อนจะเก็บจานที่เหลือเพียงแค่เศษขนมบนโต๊ะอย่างรวดเร็ว

|[ " อื้อ...ลุกขึ้นสิเจ้าสไลม์น้อย ว่าแต่เจ้าชื่ออะไรเนี่ย "] ฟีลลุกขึ้นบิดขี้เกียจก่อนจะถามอีกฝ่ายที่กำลังตื่นเต้นอยู่ |

-- [ ข้ามีนามว่า ได ข้าว่าข้าบอกท่านไปแล้วนะ ] ไดตอบด้วยใบหน้าที่เซ็งหน่อยๆ --

| [" เอ่อออ... "] ท่านราชาทำหน้าไม่รู้ไม่ชี้

[" อ้ะ มาสิ "] ฟีลทักเรียกก่อนจะเดินเปิดประตูแล้วกวักมือดันและไดให้ไปกับตน |

--  ไดเดินตามไปติดๆ --

<ดวงตาสีชมพูสวยสว่างจับจ้องเครื่องชามกระเบื้องในมือตนเล็กน้อย ก่อนที่ริมฝีปากชมพูหวานจะกระตุกยิ้มอ่อนโยนออกมา นั่นเป็นเพราะตั้งแต่ที่เธอมาเป็นภรรยาของชายหนุ่มร่างสูงโปร่งเรือนผมขาวสวยนั้น เธอไม่เคยเห็นเพื่อนของเขาเลยสักครั้ง แต่บัดนี้กลับมีฝูงเพื่อนๆเพิ่มขึ้นแบบที่เธอเองก็ตั้งตัวไม่ทัน

แต่เธอก็ไม่สนใจ เมื่อได้เห็นฟีลอยู่กับผู้คนที่ไว้ใจและเข้าใจในตัวเขา นาโอมิก็ดีใจราวกับถูกรางวัลที่หนึ่งไปแล้ว

มือนวลขาวทำความสะอาดเครื่องชามไปพลางๆ เมื่อเสร็จก็เช็ดมือจนสะอาดแล้ววานเหล่าคนรับใช้ที่เพิ่งกลับมาจากการออกไปข้างนอกให้เก็บจานชามให้ตนก่อนจะเดินไปหาฟีลและตามไปอย่างห่างๆช้าๆ

-- ไดเหลือบไปเห็นภรรยาของฟีลที่ตามมา จึงโบกมือด้วยใบหน้าอันยิ้มแย้ม ก่อนจะหันกลับไปหาท่านราชา --

| [" ไปห้องครัวกันก่อนมั้ย ? "] ฟีลกล่าวขึ้นกับทั้งสองคนเชิงถามว่าอยากไปห้องไหนก่อนด้วยใบหน้าที่ครุ่นคิดไปด้วยแต่ว่าในมือยังคงถือพายชิ้นน้อยอยู่ |

-- [ นี้ท่านยังอยากทานอยู่อีกหรอเนี่ย ] พูดด้วยน้ำเสียงกวนเท้า --

| [" ก็เรื่อยๆนะ เอามั้ย ? "] แบ่งพายในมือเป็นอีกครึ่งหนึ่งให้อีกฝ่าย

[" คิดอีกทีไม่เอาละ "] หยิบส่วนนั้นเข้าปากก่อนจะเดินเปิดประตูบานใหญ่ที่ประดับประดาไปด้วยลวดลายอันสวยงามแล้วก้าวเดินออกไปนำทั้งสองฝ่าย |

-- [ *ชิ ไอราชาหูแหลม* ] คิดในใจและทำหน้าหงุดหงิด จากนั้นก็เดินตามท่านราชาที่เดินออกไป เผยให้เห็นสวนที่ราวกับว่าที่นี้คือสรวงสวรรค์อย่างไงอย่างงั้น --

| [" อ่าาา นี่เป็นสวนหลังพระราชวังแห่งนี้ ดอกไม้ทุกดอกในสวนภรรยาข้าเป็นคนดูแลกับเมดๆเอง "] ฟีลพาอีกฝ่ายเดินไปสักพักก่อนจะพูดขึ้นแล้วผายมือไปด้านข้างของ

เผยให้เห็นสวนวิวาห์ขนาดใหญ่ไว้เดินเล่นได้สบายๆ |

-- [ ที่นี้สวยงามมากเลยครับ! ] ในขณะที่พูดอย่างตื่นเต้นก็หันซ้ายหันขวาเพื่อชมสิ่งที่น่าอัศจรรย์ที่สุดในชีวิตของเขาไปด้วย --

| [" แน่นอน ภรรยาข้าเอาใจใส่อย่างดีเลย ]" เดินนำอีกฝ่ายชมนกชมไม้ไปเรื่อยๆ |

["ทุกท่านมาที่นี่กันหรอกเหรอคะ"]

<หญิงสาวในชุดเดรสยาวสีชมพูสว่างอ่อนประดับประปรายด้วยลายลูกไม้สีครีมดูน่ารัก ใบหน้าหวานอ่อนช้อยแต่งแต้มด้วยเครื่องสำอางสีอ่อนพร้อมกับริมฝีปากสีชมพูอวบอิ่มกวาดยิ้มขึ้นบนใบหน้าเล็ก เธอเดินเหยียบสวนหย่อมหญ้าซึ่งตัวเธอเองเป็นผู้ดูแลมันกับมือ นาโอมิยิ้มที่สามีของตนพาผู้มาเยือนมาเยี่ยมเยียนสถานที่ร่มรื่นแบบนี้

| [" อื้ม.....อะ "] ฟีลพยักหน้าด้วยรอยยิ้มบางๆก่อนจะนึกบางอย่างออกก่อนจะทำหน้าซีดเบาๆ |

<ดวงตาสีชมพูสว่างมองใบหน้าของผู้เป็นสามีก็รู้ได้ เธอเม้มปากเบาๆก่อนจะเปล่งเสียงเอ่ยถามอีกคน

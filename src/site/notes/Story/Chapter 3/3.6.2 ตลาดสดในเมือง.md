#feel `feel,dai`

------------------------------
-- ไดเดินมาถึงตลาดแล้ว

[ ถึงแล้วสินะ ดูครึกครื้นจังแฮะ] ไดกล่าวก่อนจะเดินไปสำรวจร้านต่างๆ --

| [" ทำไรอยู่ที่นี่หรอ.. "] จู่ๆฟีลก็ไปสะกิดหลังไดอย่างฉับพลันโดยที่อีกฝ่ายไม่รู้ตัว |

-- [ อ่ะ...ท่านราชา ] ไดสะดุ้งตกใจหลังจากราชาเอลฟ์มาสะกิดหลัง

[ ข้ามาหาวัตถุดิบไปทำขนมน่ะครับ ] ไดกล่าวอย่างสุภาพก่อนจะเดินต่อไป --

| [" ขนมอะไรรึ? "] ฟีลเอนคอกับตัวเชิงสงสัยด้วยใบหน้าที่นิ่งขรึมกับดวงตาที่ดูเหมือนจะไร้ขีวิตชีวาแต่ก็ทำท่าทางเหมือนเป็นเด็กสดใส |

-- [ ชูครีมน่ะครับ ] ไดหันกลับไปตอบราชาเอลฟ์ --

| [" ชู...ครีม? "] ราชาทำหน้างงเพราะตั้งแต่เกิดขึ้นมายังไม่เคยได้ยินชื่อขนมแบบนั้นมาก่อน

{ในหัวของท่านราชากำลังคิดถึงไดที่กำลังยกมือขึ้นและมีครีมนมวัวอยู่บนมือแล้วให้ราชากินจากบนมือของอีกฝ่าย}

[" ?? "] |

-- [ เดี๋ยวข้าทำให้กินนะ ] ไดเดินไปพลางยิ้มให้ราชาเอลฟ์ไปด้วย

[ ว่าแต่ท่านว่างไหมละครับ ] ไดถามราชาเอลฟ์ --

| [" ว่างสิ..ทำไมหรอ... "] ฟีลกล่าวถามอีกฝ่ายเชิงสงสัยเล็กน้อยพรางจ้องมองใบหน้าอันแสนเด็กของอีกฝ่าย |

-- [ งั้นไปเดินซื้อของด้วยกันเถอะ ] ไดเดินไปจับมือราชาเอลฟ์พร้อมเดินไปที่ร้านต่างๆ --

| [" รีบจังนะ "] ปล่อยอีกฝ่ายพาตนไปไล่ซื้อของ |

-- ไดและราชาเอลฟ์เดินมาถึงร้านผลไม้ร้านหนึ่ง

[ แม่ค้าครับ มีสตรอว์เบอร์รีไหมครับ ] ไดถามแม่ค้าขณะที่เลือกดูผลไม้อื่นๆไปด้วย --

🍽️

['มีจ้าา หวานๆสดๆทั้งนั้นเลย เอาพันธุ์ไหนดีจ๊ะ พันธุ์นี้เพิ่งนำเข้าจากเมืองคนเล็กเลยนะ']

แม่ค้าวัยกลางคนร่างท้วมพูดอย่างเจื้อยแจ้ว พลางผายมือเชิญชวนเด็กหนุ่มที่ดูเหมือนกับนักเดินทางให้เข้ามาดูแผงลอยในร้านเธอเสียก่อน

-- [ นี่ๆ ท่านราชา เลือกอันไหนดีละครับ ] ไดหันไปถามราชาเอลฟ์ที่กำลังยืนกอดอกอยู่ด้านหลังเขา --

| [" ให้งูข้าเลือกดีกว่า "] ฟีลกล่าวขึ้นก่อนจะมีเลื้อยออกมาจากหลังคอก่อนจะเลื้อยไปทับสตรอว์เบอร์รี่แล้วคาบเอาแต่ลูกดีๆ

ฟีลเอื้อมมือไปหยิบลูกเหล่านั้นให้แม่ค้า |

['จ้าาา... เอ๊ะ...']

แม่ค้าเอื้อมมือจะไปรับสตรอว์เบอร์รี่พันธุ์งามของเธอ แต่ก็กลับชะงักเมื่อเห็นใบหน้าของคนที่ยื่นผลไม้ดังกล่าวมาให้เธอ

สีผมแบบนี้ หน้าหล่อๆแบบนี้ คำพูดแบบนี้

ชัดเลย...

['ราชาเสด็จมา!!!!']

ทันใดนั้นเธอก็ตะโกนแผดเสียงออกมาลั่นตลาด ท้ังตลาดเงียบเชียบ แล้วทุกคนก็ต่างแตกตื่นวิ่งเปิดทางให้ท่านราชาของพวกเขา

| [" อ้ะ?.. "] ทุกอย่างเกิดขึ้นเร็วจนฟรลประมวลผลไม่ทันก่อนจะหันหน้าไปหาได

[" ทำไงดีอ่ะ "] |

-- [ เอ่อออ...ผมว่าเรารีบจ่ายตังแล้วรีบไปที่อื่นต่อดีกว่าครับ ] ไดรับของและยื่นเงินให้กับแม่ค้าก่อนที่จะจับมือราชาวิ่งไปด้วยความรวดเร็ว 

[ อะไรของพวกเขากันนะ ทำไมเห็นราชาต้องแตกตื่นกันขนาดนั้นเลย ] ไดบ่นพรึมพรำพร้อมหอบด้วยความเหนื่อย --

-- หลังจากนั้นไดและราชาเอลฟ์ก็เดินซื้อของกันต่อ

ผ่านไปสักพักทั้งสองก็ซื้อของกันจนเสร็จ

[ แค่นี้ก็น่าจะพอแล้วล่ะ ] ไดกล่าวขณะหิ้วของมากมายไว้

[ ท่านราชา เรากลับพระราชวังกันเถอะ ] --

| ฟีลได้เดินล้วงนำหน้าไปด้วยสัมภาระแน่นเยอะแยะ |
